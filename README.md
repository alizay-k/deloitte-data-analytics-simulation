# Deloitte Data Analytics Job Simulation — Forage

This repository contains my work from Deloitte's Data Analytics Job Simulation, 
completed via Forage in July 2026. The simulation covered two areas: data 
classification using Excel and data visualization using Tableau, framed around 
a forensic technology / data analysis scenario.

This repository reflects my own analysis and approach, completed independently 
as part of the simulation. It is shared for portfolio purposes.

## What this covers

**Task 1 — Data Visualization (Tableau)**
Analyzed factory machine downtime data to identify patterns and reliability 
issues across different equipment types. Built an interactive dashboard to 
highlight machines with the highest and lowest downtime.

**Task 2 — Data Classification (Excel)**
Classified employee compensation "equality scores" across factories and job 
roles into three categories — Fair, Unfair, and Highly Discriminative — using 
a nested IF/ABS formula in Excel, to support internal pay equity analysis.
